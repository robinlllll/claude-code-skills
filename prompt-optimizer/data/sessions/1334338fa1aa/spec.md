# PromptSpec

## Goal
分析上市公司季度财报，提取关键指标并给出投资建议

## 任务一句话
基于季度财报数据，提取核心财务指标、识别业务亮点/风险、给出结构化的投资分析报告。

## 输入
- 输入格式：公司季度财报文本（包含利润表、资产负债表、现金流量表的关键数据）
- 输入示例：
  ```
  公司：XYZ科技
  季度：2024Q3
  营收：50亿元（同比+15%）
  净利润：8亿元（同比+20%）
  毛利率：35%
  经营现金流：12亿元
  资产负债率：45%
  ...
  ```

## 输出
- 输出格式：结构化分析报告，包含以下部分
  1. 关键指标摘要（表格）
  2. 业务亮点（3-5条）
  3. 风险提示（3-5条）
  4. 估值参考
  5. 投资建议（买入/持有/卖出 + 理由）

## 硬约束（必须满足）
- 必须提取：营收、净利润、毛利率、经营现金流、资产负债率
- 必须给出同比/环比变化
- 投资建议必须有明确依据，不能空泛
- 风险提示必须具体，指出触发条件

## 软约束（尽量满足）
- 语言简洁，避免废话
- 数字用表格呈现
- 对比行业平均水平（如有数据）

## 必须避免
- 不要编造数据
- 不要给出没有依据的投资建议
- 不要遗漏重要的负面信息
- 不要使用模糊表述如"表现不错"、"有所增长"

## 评估标准
- 关键指标提取完整性（5项必须全有）
- 分析逻辑是否自洽
- 投资建议是否有数据支撑
- 风险提示是否具体可验证
