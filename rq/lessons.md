# Lessons Learned

## 2026-02-14 | 2026-02-14: GPT o3 和 Grok 在回答研究问题时会大量编造外部数据源。经独立验证发现：(1) 两个模型都会用真实公司名（Similarweb、Sensor Tower、App Annie、AppsFlyer）包装完全虚构的数据点；(2) 会编造不存在的公司（如 CloudX，两个模型独立编造了同一家假公司但给出不同融资金额）；(3) 伪造精确百分比（如 CPM +18%/+14%/+21%）来增强可信度；(4) Grok 存在中文数字单位混乱（亿 vs billion）。已在 prompt 中新增数据诚信要求：禁止编造，所有外部数据必须标注来源类别（SEC/IR、新闻/行业报告、个人估算、无可靠数据）。允许引用新闻和行业报告但须注明出处。回答不设字数上限，鼓励完整详尽。
