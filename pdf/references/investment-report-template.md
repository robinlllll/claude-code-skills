# Investment Report PDF Template

## Standard Research Report Layout

```
Page 1: Cover
  - Report title (centered, large)
  - Subtitle / ticker(s)
  - Date
  - Author: "Robin's Research"
  - Classification: "CONFIDENTIAL"

Page 2+: Content
  Header: "Investment Research Report" | date | page number
  Footer: "CONFIDENTIAL" | "Generated by Claude"

Section 1: Executive Summary
  - 3-5 bullet points
  - Key thesis / conclusion
  - Target price / rating (if applicable)

Section 2: Company Overview
  - Business description
  - Key metrics table (Revenue, EBITDA, Market Cap, etc.)
  - Competitive position

Section 3: Investment Thesis
  - Bull case / Bear case
  - Key catalysts
  - Variant perception (what market is missing)

Section 4: Financial Analysis
  - Revenue breakdown (table)
  - Margin trends (chart)
  - Key ratios table

Section 5: Valuation
  - Methodology summary
  - Comparable companies table
  - DCF summary (if applicable)
  - Sensitivity analysis

Section 6: Risks
  - Key risk factors (bulleted)
  - Kill criteria

Section 7: Appendix
  - Detailed financial tables
  - Data sources
```

## Quick Report Builder

```python
def build_research_report(title, ticker, sections, output_path):
    """
    sections: list of dicts with 'heading', 'content' (str or table data)
    """
    doc = SimpleDocTemplate(output_path, pagesize=A4,
                            rightMargin=2*cm, leftMargin=2*cm,
                            topMargin=2.5*cm, bottomMargin=2.5*cm)
    story = []

    # Cover page
    story.append(Spacer(1, 8*cm))
    story.append(Paragraph(title, title_style))
    story.append(Spacer(1, 1*cm))
    story.append(Paragraph(ticker, subtitle_style))
    story.append(Spacer(1, 2*cm))
    story.append(Paragraph(datetime.now().strftime('%Y-%m-%d'), date_style))
    story.append(PageBreak())

    # Content sections
    for section in sections:
        story.append(Paragraph(section['heading'], h1_style))
        if isinstance(section['content'], str):
            story.append(Paragraph(section['content'], body_style))
        elif isinstance(section['content'], list):
            # Table data
            table = build_financial_table(section['content'])
            story.append(table)
        story.append(Spacer(1, 12))

    doc.build(story, onFirstPage=header_footer, onLaterPages=header_footer)
```
